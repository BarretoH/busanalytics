Install add-ins by downloading and then using the Add-Ins Manager to install: keyboard shortcut alt, t, i.

The latest FRED version is available in the Microsoft App Store (from Excel, Insert --> Get Add-ins), but it is buggy and error-prone so I continue to use the older version that can be freely downloaded above.

The course also uses a Bloomberg Excel add-in, but it requires access to the Bloomberg terminal, which is expensive and takes some time to learn. These additional course files are available in the Instructor Resources link at top right.
